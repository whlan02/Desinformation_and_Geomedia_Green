About GeoCam

GeoCam is part of the Desinformation and Geomedia Study project at the Institute for Geoinformatics, developing cutting-edge solutions to combat misinformation through verifiable location data.

Project Background

In an era of increasing digital manipulation and misinformation, GeoCam provides a robust solution for creating tamper-evident, location-verified photographs. Our cryptographic approach ensures that every image captured contains verifiable proof of its authenticity and location.

Goals

  • Combat photo manipulation and misinformation
  • Provide cryptographic proof of photo authenticity
  • Enable location verification for media content
  • Support journalism and research with verified media

Quick Start Examples

Android (Kotlin)

// Initialize GeoCam
val geoCam = GeoCam.Builder()
    .enableLocationVerification(true)
    .setSigningKey(keyPair)
    .build()

// Capture verified photo
geoCam.takePictureAsync { result ->
    when (result) {
        is Success -> handleVerifiedImage(result.image)
        is Error -> handleError(result.exception)
    }
}

iOS (Swift)

// Initialize GeoCam
let geoCam = GeoCam.Builder()
    .enableLocationVerification(true)
    .setSigningKey(keyPair)
    .build()

// Capture verified photo
geoCam.takePictureAsync { result in
    switch result {
    case .success(let image):
        handleVerifiedImage(image)
    case .failure(let error):
        handleError(error)
    }
}

Rust CLI

# Verify a GeoCam image
geocam verify --image photo.jpg --public-key key.pub

# Batch verify multiple images
geocam batch-verify --directory ./photos --output report.json

# Extract metadata from GeoCam image
geocam extract --image photo.jpg --format json

Frequently Asked Questions

How does GeoCam prevent location spoofing?

GeoCam uses multiple validation techniques including GPS signal analysis, sensor fusion, and cryptographic timestamps to detect and prevent location spoofing attempts.

What image formats are supported?

GeoCam supports JPEG and PNG formats. The metadata is embedded using steganographic techniques that preserve image quality.

Can I verify images taken with other apps?

Only images captured with GeoCam can be cryptographically verified, as the signing process occurs during capture.
